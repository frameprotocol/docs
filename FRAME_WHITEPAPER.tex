\section*{Conversion Instructions}

This \LaTeX{} file provides the document preamble, styling, and title page for the FRAME whitepaper. To generate the complete PDF:

\subsection*{Option 1: Pandoc (Recommended)}
\begin{lstlisting}[language=bash]
# Install pandoc and a LaTeX distribution
sudo pacman -S pandoc texlive-core  # Arch Linux
# or: sudo apt install pandoc texlive-xetex  # Ubuntu/Debian

# Generate PDF from Markdown
pandoc FRAME_WHITEPAPER.md -o FRAME_WHITEPAPER.pdf \
  --pdf-engine=xelatex \
  -V geometry:margin=1in \
  -V fontsize=11pt \
  --toc \
  --number-sections \
  --highlight-style=tango
\end{lstlisting}

\subsection*{Option 2: Full LaTeX Compilation}
\begin{lstlisting}[language=bash]
# Convert Markdown body to LaTeX
pandoc FRAME_WHITEPAPER.md -o body.tex --no-highlight

# Insert body.tex content into this file (replacing this section)
# Then compile:
xelatex FRAME_WHITEPAPER.tex
xelatex FRAME_WHITEPAPER.tex  # Run twice for TOC
\end{lstlisting}
